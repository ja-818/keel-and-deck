Capabilities

Everything agents need to be great

Not just a chat window. Each agent has its own workspace, memory, skills, and connections.

Skills & memory

Agents learn from every interaction. Teach them once, they remember forever. Install community skills for new capabilities.

Routines

Schedule recurring tasks. Your agent checks in every morning, monitors metrics, or summarizes overnight activity. On autopilot.

Workspace files

Each agent has its own workspace with files, instructions, and context. Everything it needs, organized in one place.

Slack sync

Connect Slack with one click. Each conversation becomes a thread. Reply from your phone, your agent responds.

Runs locally

Desktop app on your Mac. Your data stays on your machine. No cloud required for the base experience.

Open source

MIT licensed. The full codebase, the framework, and the agent store. Fork it, extend it, build on it.
