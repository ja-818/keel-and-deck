# Prepare a Handoff to a Lawyer

User-facing entry point for `draft-a-legal-document` (escalation brief).

## Talking to the user

Plain language. Confirm the brief is ready and explain what to do with it ("Email this to an outside lawyer. I can suggest the kind of firm, but you'll pick the actual one."). Never name files or paths.
