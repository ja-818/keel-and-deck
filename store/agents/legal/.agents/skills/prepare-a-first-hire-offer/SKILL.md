# Prepare a First-Hire Offer

User-facing entry point for `prepare-a-job-offer`. Built for the very first employee, where founders typically need the full packet (offer letter + CIIAA + option grant + exercise agreement) at once.

## Talking to the user

Plain language. Confirm the four pieces are ready in draft form, that nothing is signed or sent, and remind them of the 30-day 83(b) window if the hire early-exercises. Never name files or paths or the internal procedure.
