# Auto-Answer an Enterprise Security Questionnaire

User-facing entry point for `answer-a-security-questionnaire`.

## Talking to the user

Plain language. Lead with how many of the questions are already answered and how many need the user. Group the user's pending questions by topic. Never name files or paths. Don't fabricate answers.
