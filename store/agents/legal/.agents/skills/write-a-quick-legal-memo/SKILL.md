# Write a Quick Legal Memo

This is the user-facing entry point for `answer-a-legal-question`. It collects the founder's question and runs the full memo procedure.

## Talking to the user

Send the answer in plain language. Bottom line first, then the reasoning. Never name the underlying procedure or the file the memo is saved to. If you need more from the user, ask one short, plain-English question.
