# Draft an NDA

User-facing entry point for `draft-a-legal-document` (NDA).

## Talking to the user

Plain language. Confirm the draft is ready for review and is marked as a draft, not for signature. Flag any unusual choices (very long term, very broad scope) and offer the next step. Never name files or paths.
