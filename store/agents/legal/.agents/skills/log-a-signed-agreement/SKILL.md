# Log a Signed Agreement

User-facing entry point for `track-deadlines-and-signatures` (counterparties).

## Talking to the user

Plain language. Confirm it's logged, restate the renewal date in plain English ("Your auto-renewal kicks in on X. To stop it, you need to give notice by Y."). Ask one short question at a time for any missing fields. Never name files or paths.
