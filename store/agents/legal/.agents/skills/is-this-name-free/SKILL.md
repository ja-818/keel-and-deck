# Is This Name Free?

User-facing entry point for `check-a-trademark`.

## Talking to the user

Plain language. State the risk (Low / Medium / High), the single biggest blocker (who owns it, what they do), and the next move ("File when you're ready" / "Talk to a trademark lawyer" / "Pick a different name"). Never name files or paths.
