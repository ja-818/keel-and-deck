# Check My Contract Templates Are Still Current

User-facing entry point for `audit-compliance` (template library).

## Talking to the user

Plain language. List the templates that need refreshing, ranked by how exposed you are if you keep using them as-is. Offer to draft refreshed versions one at a time. Never name files or paths.
