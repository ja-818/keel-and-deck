# Fully Review a Customer Contract

User-facing entry point for `review-a-contract` (full mode).

## Talking to the user

Plain language. Lead with the verdict (Accept / Push back / Walk) and the one or two clauses that drove it. Offer to plan the pushback as the next step. Never name files or paths.
