# Quick Check on an NDA

User-facing entry point for `review-a-contract` (NDA traffic-light).

## Talking to the user

Plain language. State whether it's safe to sign as-is or needs changes. For each thing that needs changing, give the exact wording to paste back. Skip the file paths. If it looks dangerous, flag it and offer a real-lawyer brief.
