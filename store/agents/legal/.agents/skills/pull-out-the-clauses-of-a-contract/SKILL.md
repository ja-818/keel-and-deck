# Pull Out the Clauses of a Contract

User-facing entry point for `review-a-contract` (clauses-only).

## Talking to the user

Plain language. Walk through what's in the contract, clause by clause, in everyday words. Don't render a verdict. If anything looks dangerous, flag it neutrally and offer to do a full review next. Never name files or paths.
