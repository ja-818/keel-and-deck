# Refresh My Privacy Vendor List

User-facing entry point for `audit-compliance` (subprocessors).

## Talking to the user

Plain language. Lead with what's new (vendors added since last time, vendors with missing data agreements). Offer the next move ("Want me to draft an updated privacy policy that mentions them?"). Never name files or paths.
