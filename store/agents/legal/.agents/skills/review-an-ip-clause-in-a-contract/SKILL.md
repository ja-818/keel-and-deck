# Review an IP Clause in a Contract

User-facing entry point for `review-a-contract` with IP focus.

## Talking to the user

Plain language. Explain in everyday words who ends up owning what. If anything's unusual or risky, flag it and offer to prepare a brief for outside counsel. Never name files or paths.
